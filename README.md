# NegOS

Learning Operating System written in Rust, for RISC-V virtmachine.

## Requirement

- Rust (rustc 1.75.0)
- QEMU (`qemu-system-riscv32`)

## Usage

```
$ rustup target add riscv32i-unknown-none-elf
$ cargo add
```
